# Web
## HTML: 负责网页的结构（页面元素和内容）。
## CSS:负责网页的表现（页面元素的外观、位置等页面样式，比如:颜色、大小等） 
## JavaScript:负责网页的行为（交互效果）
# 什么是HTML?
## HTML（HyperText Markuo Language）:超文本标记语言
## 超文本就是超越了文本的限制，比普通的文本更强大，不仅包含文字信息，还可以定义图片、音频、视频等内容
## 标记语言就是由 标签 "<标签名>" 构成的语言
# 什么是:CSS?
## CSS(Casacding Style Sheet) :层叠样式表，用于控制页面的样式表现;
